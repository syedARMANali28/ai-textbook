# Chapter 1: The Core Concepts of ROS 2 - Quiz

**1. What is the primary role of ROS 2 in robotics?**
   a) To replace all robot hardware.
   b) To provide a framework for writing robot software and facilitating communication between components.
   c) To develop highly specialized AI algorithms only.
   d) To design the physical structure of robots.

**2. What is the fundamental unit of computation in ROS 2?**
   a) A topic.
   b) A service.
   c) A node.
   d) An action.

**3. Which of the following is NOT a common communication pattern in ROS 2?**
   a) Topics.
   b) Services.
   c) Emails.
   d) Actions.

**4. What characteristic best describes ROS 2 topics?**
   a) Synchronous, one-to-one communication.
   b) Asynchronous, one-to-many broadcast.
   c) Long-running, goal-oriented tasks.
   d) Request-response model.

---
**Answers:**
1. b)
2. c)
3. c)
4. b)
