# Chapter 2: Communication Patterns - Quiz

**1. Which ROS 2 communication pattern is best suited for streaming continuous sensor data?**
   a) Services.
   b) Actions.
   c) Topics.
   d) Nodes.

**2. What is a key characteristic of a ROS 2 Service?**
   a) It's asynchronous and one-to-many.
   b) It provides continuous feedback for long-running tasks.
   c) It's synchronous and one-to-one, requiring a response.
   d) It's a broadcast system without guaranteed delivery.

**3. When would you typically use a ROS 2 Action?**
   a) To get an immediate response to a single query.
   b) To send a constant stream of data.
   c) For long-running, goal-oriented tasks that require feedback and possible cancellation.
   d) To define the physical structure of a robot.

**4. If a client needs a guaranteed response from a server for a short-lived operation, which communication method should be used?**
   a) Topics.
   b) Services.
   c) Actions.
   d) Nodes.

---
**Answers:**
1. c)
2. c)
3. c)
4. b)
