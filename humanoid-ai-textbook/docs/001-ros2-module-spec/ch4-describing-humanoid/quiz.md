# Chapter 4: Describing Your Humanoid - Quiz

**1. What does URDF stand for?**
   a) Unified Robotics Definition File
   b) Universal Robot Description Format
   c) Unified Robot Description Format
   d) Utility Robot Design Framework

**2. Which URDF element describes the rigid bodies of a robot?**
   a) `<joint>`
   b) `<transmission>`
   c) `<link>`
   d) `<actuator>`

**3. What is the purpose of a `<joint>` element in URDF?**
   a) To define the robot's power source.
   b) To describe the connection and movement between two links.
   c) To specify the robot's visual appearance only.
   d) To define the robot's control algorithms.

**4. What is RViz primarily used for in the context of URDF?**
   a) Writing URDF files.
   b) Compiling URDF files.
   c) Visualizing URDF models and other ROS 2 data in a 3D environment.
   d) Controlling physical robot hardware directly.

---
**Answers:**
1. c)
2. c)
3. b)
4. c)
