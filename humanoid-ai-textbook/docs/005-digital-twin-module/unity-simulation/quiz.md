# Unity Simulation Chapter Quiz

**1. What is Unity primarily known for, which also contributes to its strength in robotics simulation?**
   a) Its open-source physics engine.
   b) Its high-fidelity graphics and game development capabilities.
   c) Its tight integration with ROS.
   d) Its focus on industrial automation.

**2. For which robotics application is Unity's visual realism particularly valuable?**
   a) Testing precise bipedal locomotion.
   b) Training computer vision models and human-robot interaction.
   c) Simulating low-level sensor noise.
   d) Validating complex rigid body dynamics.

**3. What programming language is primarily used for scripting in Unity?**
   a) Python
   b) C++
   c) C#
   d) Java

**4. How does Unity's ability to generate synthetic datasets benefit AI development?**
   a) By requiring more manual data annotation.
   b) By producing perfectly labeled data with visual realism, eliminating manual annotation.
   c) By focusing solely on abstract data.
   d) By only simulating simple 2D environments.

---
**Answers:**
1. b)
2. b)
3. c)
4. b)
